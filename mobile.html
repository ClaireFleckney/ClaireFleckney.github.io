Dungeon Escape Mobile Game

Created as the Final Project for my Mobile App Development class, this is a simple roguelike dunger crawling game. It features simple character creation, randomized encounters, and one of four randomly chosen maps. It was created using React Native and the Expo development and deployment tools, and was released for Android devices.
